Applications of SVD

Examples of real-life applications of SVD and PCA


Unsupervised Machine Learning

SVD is widely used in many unsupervised machine learning algorithm techniques. It is used for building recomendation system as well as face recognition.

Dimensionality Reduction

Visualizing the structure and pattern that exists within the high-dimensional data can be challenging. PCA can help project the high-dimensional data onto a plane in 3D or 2D, thus enabling one to visualize the data better.

Image Compression and Signal Processing

SVD can sucessfully compress image into smaller size by applying SVD to the pixel matrix. In doing so, one can obtain the principle components of the matirx. Picking only a subset of all the principle components can reduce the size of the image while minimizing the possible loss of information from image compression
